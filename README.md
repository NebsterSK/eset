# Common assets for LWS projects

## Prerequisites

- PHP 8.1 or higher
- Laravel 10

## Installation

Install the package via Composer:

```bash
composer require nebstersk/eset
```

## Usage

Publish common assets / files with:

```bash
php artisan eset:publish
```
